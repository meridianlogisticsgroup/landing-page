A single stack for booking, estimates, dispatch, and profitability—built for moving companies that operate at scale.

  • Per-km pricing via Google Maps API with transparent tolls & fuel assumptions
  • Resource batching (e.g., Medium Move = 15-ft truck + 3 movers)
  • Nearest-dispatch + live truck availability
  • Quotes → Jobs → Invoices in one flow
  • Automated cost/expense/profit tracking
  • Scheduling, utilization reporting, exportable analytics
# Pseudocode – pricing & batching
distance = gmaps.route_km(pickup, dropoff)
batch = catalog["medium_move"]  # truck + crew
base = batch.rate_base + (distance * batch.rate_per_km)

if is_peak_window(date):
  base *= 1.15  # peak factor

quote = tax.apply(base, region)
assign_to = dispatch.nearest_available_truck(pickup, date)
reserve(assign_to, crew=batch.crew, window=date.window)
